

## 特性
- 有驱动列表可以选择并自动下载，免除痛苦
- tables只展示 超级表，虚拟表，普通表。不显示 子表（没啥意义）
- 视图还在适配中（没有企业版无法验证）
- 方言支持（难度太大，等一等吧）
